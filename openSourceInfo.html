Can I trust the Melatect diagnosis tool?

We always strongly recommend users confirm any diagnoses produced by the Melatect app with a medical professional with dermatology experience. To streamline this process, users can locate a nearby dermatologist within the app in the "My Doctors" page found in the bottom tab bar.

Developers at Tonnelier Technologies are not responsible for misdiagnoses generated by the Melatect app, and we suggest reading our disclaimers page before using the Melatect melanoma diagnosis tool.

What database was used to make the Diagnosis feature?

The Diagnosis feature was created using a compilation of over 30,000 augmented and original malignant/benign images. All malignant images were curated by developers from local dermatology clinics as well as the massive ISIC Skin Cancer Database. ISIC is a reliable resource created by medical professionals. Some benign images used in creating the diagnosis tool were collected from people using a simple iPhone camera, while most were also collected from accredited dermatology clinics and the ISIC database.

How does Melatect utilize user data?

Because the current Melatect version is BETA, Melatect does not collect any data. Users cannot make profiles within the app as of yet (that functionality is coming soon!) and therefore, NO data is collected from any users. When profiles become a possibility, basic data (such as emails and names) will be stored on a HIPPA compliant database where data cannot be accessed by any developers or company personnel. Any photos taken within the app are stored locally within user's phones rather than via a database and will remain so in the future. Please contact us at tonnelierco@gmail.com with any further concerns regarding Melatect data collection.

Does Melatect sell user data?

Nope. Tonnelier Technologies does not sell user data collected by any of its apps, including Melatect, and will not sell any user data in the future. For more information on how data is collected, please visit our Privacy Policy.
